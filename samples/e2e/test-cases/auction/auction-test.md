# 第一部分：提交出价功能

---

## 用例1: 正常提交出价
**编号**: `auction_bid_normal_1`

### 步骤
1. 使用 buyer001/123456 登录系统
2. 导航到艺术品详情页面（artworkId=1）
3. 查看当前最高出价（假设为 10000）
4. 在出价输入框输入：12000
5. 点击"提交出价"按钮
6. 等待响应（2秒）

### 验证
- 显示"出价成功"提示
- 出价记录被创建
- 网络请求 POST /api/auction/bid 返回 200
- 响应包含出价记录详情（bidId、金额、时间）

### 清理
- 调用 DELETE /api/auction/bid/{bidId} 撤销出价
