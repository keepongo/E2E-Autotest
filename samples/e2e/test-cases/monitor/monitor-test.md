# 监控管理功能测试

**优先级**: P1
**模块**: 监控管理
**前置条件**:
- 应用服务已启动（后端 http://localhost:8080，前端 http://localhost:3000）
- 使用管理员（admin/123456）或安保人员（security001/123456）登录
- 数据库有监控设备测试数据
**预计耗时**: 200秒

---

---

## 用例1: 查看监控设备列表
**编号**: `monitor_device_list_1`

### 步骤
1. 使用 admin/123456 登录系统
2. 导航到监控中心页面
3. 等待设备列表加载（2秒）
4. 查看设备展示

### 验证
- 页面显示监控设备列表
- 每个设备显示：设备编号、名称、区域、状态
- 显示设备在线/离线状态
- 网络请求 GET /api/monitor/device/list 返回 200

### 清理
- 无需清理
